# 킨코스 비즈 섹션 이미지

이 디렉토리는 킨코스 비즈 섹션에 사용되는 이미지들을 포함합니다.

## 필요한 이미지들

1. `biz-card-1.png` - 첫 번째 카드 이미지 (믿고 맡기는 B2B 인쇄 파트너)
2. `biz-card-1-bg.png` - 첫 번째 카드 배경 이미지
3. `biz-card-2.png` - 두 번째 카드 이미지 (비즈니스 효율을 높이는 인쇄 솔루션)
4. `biz-card-2-mask.png` - 두 번째 카드 마스크 이미지
5. `biz-card-3.png` - 세 번째 카드 이미지 (업종별 전문가가 제안하는 인쇄 전략)
6. `biz-card-3-bg.png` - 세 번째 카드 배경 이미지
7. `biz-card-4.png` - 네 번째 카드 이미지 (Kinko's for Business)
8. `biz-bg-ellipse.svg` - 배경 원형 요소

## Figma에서 이미지 추출 방법

1. Figma에서 해당 이미지들을 선택
2. 우클릭 → "Export" 선택
3. PNG 또는 SVG 형식으로 내보내기
4. 적절한 해상도로 설정 (최소 2x 해상도 권장)
5. 이 디렉토리에 저장

## 이미지 최적화

- PNG 이미지는 WebP 형식으로 변환 고려
- SVG는 코드 최적화 후 사용
- 모든 이미지는 반응형 디자인을 고려하여 다양한 해상도 제공
